# Меню и разделение ролей

## Админ-меню
- `👥 Клиенты`
- `➕ Добавить клиента`
- `📊 Аналитика`
- `🗂 Архив`

## Карточка клиента
Показывает:
- имя;
- Threads username;
- Telegram username;
- подключён / не подключён;
- таблица подключена / нет;
- контент-план подключён / нет;
- тема создана / нет;
- статус проекта;
- дата создания;
- последние результаты.

Кнопки:
- `✏️ Изменить`
- `📊 Подключить таблицу`
- `📄 Подключить контент-план`
- `🔗 Получить invite-ссылку`
- `💬 Написать клиенту`
- `🧵 Создать / проверить тему`
- `📈 Внести статистику`
- `📊 Аналитика клиента`
- `🔄 Обновить карточку`
- `🔴 Закрыть проект`

## Клиентское меню
- `📅 Ветки`
- `📄 Контент-план`
- `📊 Результаты`
- `💬 Связь с менеджером`

## Ограничение доступа
Каждый обработчик обязан проверять роль.

Клиент:
- получает только собственные данные;
- не может открыть админские callback-команды;
- не видит рабочую тему;
- не видит внутренние комментарии.

Администратор:
- имеет доступ ко всем клиентам;
- видит аналитику;
- вносит недельную статистику;
- управляет архивом.

Менеджер:
- взаимодействует через рабочую тему;
- не получает клиентское меню;
- не получает полный админ-доступ без ADMIN_ID.
