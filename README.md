# Threads Client Center 2.0

Чистая версия бота.

## Google-таблица клиента
Столбцы:
Дата | Время | Ветка | Статус

Бот отправляет только строки с сегодняшней датой и статусом `Готово`.

## Автоотправка
Каждый день в 09:00 по `TIMEZONE`.
Повторная автоотправка одной и той же ветки блокируется через `delivery_log`.

## Railway
1. Создать новый сервис.
2. Добавить Volume.
3. Mount path: `/data`
4. Добавить `DB_PATH=/data/bot.db`
5. Добавить остальные переменные из `.env.example`
6. Загрузить проект в GitHub.
7. Дождаться Online.

## Google Sheets
Сервисному аккаунту Google дать доступ к каждой таблице клиента.
JSON-ключ хранить только в Railway Variable `GOOGLE_SERVICE_ACCOUNT_JSON`.


## Постоянная связь с клиентом

Добавьте `WORK_GROUP_ID` — ID рабочей Telegram-группы с включёнными темами.

После подключения клиента бот создаёт для него персональную тему.

- Любое обычное сообщение клиента в личный бот автоматически копируется в его тему.
- Поддерживаются текст, фото, документы, видео, голосовые и другие сообщения, которые Telegram Bot API разрешает копировать.
- Ответ команды внутри темы автоматически копируется клиенту в личный бот.
- Кнопка «💬 Связь с менеджером» теперь только объясняет клиенту, как писать; одноразовый режим больше не используется.
