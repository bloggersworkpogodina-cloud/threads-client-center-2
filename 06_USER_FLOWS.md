# Пользовательские сценарии

## Создание клиента
Админ:
1. Нажимает `Добавить клиента`.
2. Вводит имя.
3. Вводит Threads username.
4. Вводит Telegram username.
5. Проверяет карточку.
6. Нажимает `Создать`.

Бот:
1. Проверяет дубль.
2. Создаёт клиента.
3. Создаёт тему.
4. Генерирует invite-ссылку.
5. Показывает карточку.

## Подключение клиента
Клиент:
1. Открывает invite-ссылку.
2. Нажимает Start.

Бот:
1. Привязывает telegram_id.
2. Показывает клиентское меню.
3. Пишет событие в тему.

## Получение веток
Клиент:
1. Получает автоматическую отправку или нажимает `Ветки`.
2. Получает каждую ветку отдельным сообщением.

## Подтверждение публикации
Клиент:
1. Получает вечерний вопрос.
2. Выбирает результат.
3. При необходимости вводит количество.

Бот:
1. Сохраняет результат.
2. Дублирует его в тему.
3. Обновляет аналитику.

## Результаты
Клиент:
1. Раз в 2 дня получает вопрос.
2. Вводит отклики.
3. Вводит заявки.
4. Добавляет комментарий.

## Общение
Клиент:
1. Нажимает `Связь с менеджером`.
2. Пишет сообщение.

Менеджер:
1. Получает сообщение в теме.
2. Отвечает в той же теме.

Бот:
1. Передаёт ответ клиенту.
