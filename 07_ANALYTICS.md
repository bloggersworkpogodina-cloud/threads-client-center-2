# Аналитика

## Показатели дисциплины
- отправлено веток;
- опубликовано веток;
- процент выполнения;
- дни без публикаций;
- частичные публикации.

Формула:
`опубликовано / отправлено × 100%`

## Показатели результата
- отклики;
- заявки;
- конверсия отклик → заявка;
- динамика по двухдневным периодам.

## Недельная статистика
- просмотры;
- лайки;
- ответы;
- репосты;
- цитирования;
- подписчики;
- переходы в Telegram.

## Выводы для администратора
Система должна помогать различать:
- низкая эффективность контента;
- низкая дисциплина публикаций;
- хорошие охваты без заявок;
- заявки при небольших охватах;
- положительная или отрицательная динамика.

## Представление
По каждому клиенту:
- текущая неделя;
- предыдущая неделя;
- изменение в процентах;
- история по периодам.

Общая аналитика:
- активные клиенты;
- средняя дисциплина;
- суммарные отклики;
- суммарные заявки;
- клиенты с ростом;
- клиенты без результата.
